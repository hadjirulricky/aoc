# The AOC Archives

## Contribution

1. Create a directory on where you'll put your solution with the format `{year}/day{day}/solutions/{{name | username | any }}`:

```
2019
  - day-1/
    - solutions/
      - isotopeee/
        - index.js
        - extrafile.js
        - util.js
```

2. Create PR.

ALL PRs are welcome ❤️
